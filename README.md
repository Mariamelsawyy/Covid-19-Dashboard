# Covid-19-Dashboard
Global COVID-19 Dashboard is an interactive web application built with Dash and Plotly that visualizes COVID-19 trends by country. Users can select a country, choose a case metric (Confirmed, Deaths, or Recovered), and filter by date range to explore dynamic time-series insights.

🌍 Global COVID-19 Dashboard (Dash + Plotly)

An interactive web dashboard built with Dash, Plotly, and Dash Bootstrap Components to visualize global COVID-19 trends over time.

The dashboard allows users to explore confirmed, death, and recovered cases by country with dynamic filtering and real-time updates.

📊 Project Overview

This project uses the COVID-19 Clean Complete Dataset to create an interactive data visualization dashboard. Users can:

Select a country

Choose a case metric (Confirmed, Deaths, or Recovered)

Filter by a custom date range

View trends in an interactive time-series line chart

The dashboard updates automatically using Dash callbacks, providing a smooth and responsive user experience.

🚀 Features

🌎 Country-based filtering

📈 Time-series visualization using Plotly

📅 Interactive date range selection

🔘 Metric selection (Confirmed / Deaths / Recovered)

🌙 Dark theme UI using Bootstrap

⚡ Real-time updates with Dash callbacks

🛠️ Technologies Used

Python

Dash

Plotly Express

Dash Bootstrap Components

Pandas

🧠 How It Works

The dataset is loaded and cleaned using Pandas.

Users select filters (country, metric, date range).

A Dash callback dynamically filters the dataset.

Plotly generates an updated line chart based on user selections.
